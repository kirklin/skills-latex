# resume — 中文一页专业简历模板

单文件、零外部 cls 依赖的中文一页简历。示例内容以公众人物（埃隆·马斯克）的
**公开履历事实**排成——与 `neurips-paper` 复刻已发表论文同理，成品即是完成度
参照；联系方式为占位符，页脚印有样例声明。

</p>

版式要点（经典中文专业版式 · 全黑 · 宋黑搭配）：

- 头部：左侧姓名区（姓名 / 字距英文名 / 定位语 / 联系方式），右上角
  2.5cm×3.5cm 证件照框位（细框占位，一行注释换真实照片）；头部与正文
  之间不加线；
- 条目**一行三列**：公司居左、职位居中、时间居右（「年.月」格式）；
  要点用中文编号「1、2、」；单行条目收纳早期经历；教育行同为三列；
- **宋黑搭配**（中文印刷排版的标准做法）：正文宋体，标题、公司、职位与
  强调用黑体——中文的"加粗"本质是宋→黑字体切换，比伪粗更专业；
  关键数据与关键词用 `\hl{}` 黑体强调；
- 全黑无灰、无彩色无图标；
- LaTeX 功力在细节：时间列 OpenType 等宽数字右缘逐位对齐、microtype 光学
  边缘、xeCJK 中西文自动间距与全角标点挤压、英文名 LetterSpace 字距、
  全文禁断词 + \emergencystretch；
- 中文宋体/黑体由 ctex 按操作系统自动选择，西文 Source Sans Pro
  （TeX Live 自带），跨平台开箱即用。

## 文件结构

| 文件 | 作用 |
|---|---|
| `main.tex` | 唯一源文件 — 版式宏（`\rsection`/`\entry`/`\entryline`/`\hl`）+ 示例内容 |

## 改成你的简历

1. 头部：姓名、英文名、定位语与联系方式在左侧姓名区直接改。
2. 正文按宏填写：
   - `\rsection{节标题}` — 节标题 + 细黑通栏线；
   - `\entry{公司/学校}{职位/专业}{时间}` — 一行三列主条目，下接 `enumerate`
     写「1、2、」编号要点（关键数据用 `\hl{}` 黑体强调）；
   - `\entryline{公司}{一句话职责或结果}{时间}` — 单行条目（早期经历、奖项）。
3. 证件照：按头部注释将 `\fbox{...}` 占位框整块替换为
   `\includegraphics[width=2.5cm, height=3.5cm]{photo.jpg}`；不需要照片时
   删除右栏 minipage 并将左栏宽度改回 `\textwidth`。
4. 要点写法：动词开头 + 量化结果，参照示例；一页放不下优先删内容而非压行距。
5. 交付前删除页脚的样例声明行。

## 编译

```bash
bash scripts/compile.sh templates/resume/main.tex --preview
```

`compile.sh` 检测到 ctex 自动使用 XeLaTeX。

## 常用调整

- **英文简历**：文字直接替换即可（版式宏与字体设置中西文通用）。
- **两页**：内容确需两页时删除 `\pagestyle{empty}` 加回页码。
- **行距/密度**：类选项 `linespread=1.3` 与列表 `itemsep` 是主要旋钮。
