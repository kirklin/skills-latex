# cn-everyday — 日常一页纸文书

普通人日常最高频的五种凭证与证明，每份一页，开箱即用：

| 文件 | 文书 | 什么时候用 |
|---|---|---|
| `iou.tex` | 借条 | 借钱给别人/向别人借钱时写，民间借贷最重要的凭证 |
| `receipt.tex` | 收条 | 收到还款、货款、押金时出具 |
| `employment-certificate.tex` | 在职证明 | 办签证、贷款、租房、子女入学 |
| `income-certificate.tex` | 收入证明 | 银行贷款、信用卡、签证（含核实电话） |
| `power-of-attorney.tex` | 授权委托书 | 委托他人代办社保、公积金、证件等事务 |

样张按高配基调写：在职/收入证明的主角是公司首席执行官（千万级年薪、
含收入构成明细与董事会聘任要素），条款话术按律师起草标准写全（用途限定、
有效期、权利保留、管辖与送达、排除事项）——模板要展示全量要件，普通场景
删繁即可。共用 `everyday.sty`。编译任意一份：

```bash
bash scripts/compile.sh templates/cn-everyday/iou.tex --preview
```

## 要件都写全了（照着换内容即可）

- **借条**：出借人/借款人全名＋身份证号、金额**大写＋小写**、交付方式与
  日期、还款期限、利息约定（未约定即视为无息）、逾期责任（含实现债权
  费用由借款人承担）、签名＋按手印＋日期
- **收条**：款项性质（还款/货款/押金）、对应哪笔往来、**尚欠金额**
- **两份证明**：都带"仅用于××用途、不作担保依据"的限定句和经办核实
  电话；单位落款处有公章 PNG 叠印钩子（`\edSeal` 内注释）
- **委托书**：委托事项逐项列明、权限边界、期限、**无转委托权**声明；
  仅适用于一般民事代办，诉讼委托请用法院格式

## 版式与宏

黑体二号居中标题（二字标题用 `\hspace{1\ccwd}` 加宽）、正文宋体小四
1.5 倍距首行缩进、无页眉页脚（单页文书惯例）。宏：`\edTitle` 标题、
`\edblank[宽]{内容}` 填写线、`\edSign[线宽]{标签}{线上内容}` 右对齐签署行、
`\edLine` 右对齐落款行、`\edSeal` 盖章位、`\hl` 加粗。

## 样例的彩蛋

借条与收条和本仓库法律模板族是**同一个虚构案件**：借条（2024-03-01 借
200,000 元、2025-02-28 到期、实现债权费用条款）→ 收条（部分还款 20,000
元、尚欠 180,000 元）→ 催告（`cn-legal-opinion/demand-letter.tex`）→
起诉（`cn-litigation/complaint.tex`），四处日期金额与条款依据完全勾稽，
可当一套"民间借贷全流程文书"示例使用。
