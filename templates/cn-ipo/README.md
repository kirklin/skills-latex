# cn-ipo · A股首次公开发行招股说明书模板

产出一份完整的 A 股 IPO 招股说明书（申报稿），框架逐条对应中国证监会
《公开发行证券的公司信息披露内容与格式准则第 57 号——招股说明书》
（证监会公告〔2023〕26 号，全面注册制配套规则），版式对标头部券商投行
申报文件，并已**用准则官方原文和真实科创板申报稿逐项对照校验**（见文末
「对照校验记录」）。示例样张为虚构的「示例科技股份有限公司」——一家
人工智能大模型公司（大模型解决方案 / 开放平台 MaaS / 智能应用）的科创板
申报稿，66 页、十二节及附件系列俱全。

```bash
bash ../../scripts/compile.sh main.tex --preview
```

## 怎么改成你的项目

1. **换主体**：`main.tex` 顶部的 `\renewcommand` 块集中了公司名、英文名、
   注册地址、板块、稿别、保荐人、签署日期——页眉、封面、概况表、签字页全部
   联动。
2. **换板块**：
   - 主板/创业板：改 `\gsListingLine`；封面的科创板风险提示段按板块要求
     替换或删除（`cover.tex` 内有注释）；第二节「科创板定位」「上市标准」
     相应调整。
   - 注册稿/正式稿：`\gsStage` 改「（注册稿）」或置空；删除封面顶部的
     预披露提示段；把全书「【】」留空项填实。
3. **换页码**：申报文件卷册页码「1-1-N」由 `\gsFascicle{1-1}` 控制，
   置空 `{}` 即纯数字页码。
4. **治理结构**：样张按新《公司法》采用「不设监事会＋审计委员会履职＋
   职工代表董事」的 2026 年申报实践；如发行人仍保留监事会，需在第四、八、
   九、十一节自行恢复监事相关表述与签字栏。
5. **填内容**：每节一个文件，注释标明了对应准则条文与数据勾稽点；
   董监高简历、分年客户明细等处已留「按同格式补全」提示。
6. **签章**：多行签名阵列用 `\sigblock{标签}{…}` ＋ `\sigzrow{姓名}{姓名}{姓名}`
   （标签居左列、签名线阵列靠右区）；单人/双人签名用 `\sigline{标签}{姓名}{姓名}`
   （标签紧贴签名线左侧、整组靠右）；电子章图片钩子见 `zhaogushu.sty` 中
   `\sigseal` 的注释。

## 版式细节（对标真实申报文件）

- 正文宋体小四、Word 1.5 倍行距（基线距 23.3pt，真实申报稿实测；比
  LaTeX 惯用的 1.5×21.6pt 更疏朗）、首行缩进两字；西文/数字自动使用
  Times New Roman（未安装则回退）；CJK 字体由 ctex 按操作系统选择
  （Windows 下即中易宋体/黑体，与真实申报 PDF 一致）。
- 标题（字号、缩进与段前段后均按 2026 年真实申报稿逐项实测校准）：
  `第X节` 黑体三号居中（自动中文编号，`\gssection` 起新页）；`一、`
  黑体四号顶格；`（一）` 黑体小四**缩进两字**；`1、` `（1）` 宋体加粗
  小四缩进两字，由 `\subsection`–`\subparagraph` 自动编号。
- 表格：全线框（Word 风格）、表头加粗居中（`\thA/\thB`，长表头用
  `\thAw/\thBw{宽}` 自动换行）、单元格垂直居中、金额右对齐、
  `\biaodw{万元}` 在表右上角标注单位、`\tabnote{…}` 表下注、
  longtable 跨页自动续排表头。
- 财务表列序遵循申报惯例：最新报告期在前；行文叙述按时间顺序。
- PDF 书签与目录联动（hyperref，链接无框无色）。

1. **准则第 57 号官方原文**（上交所网站刊载版）：十二节框架、第十五至十九条
   封面/扉页/目录/释义要求、第二十一条概览声明原句、第二十二条概览参考格式
   （中介机构三表、发行概况表、重要日期表）、第二十三条风险三分类、第二十七
   条风险披露边界、第八十五至九十条各方声明措辞及签字人（含项目协办人）。
2. **真实科创板申报稿（2023 年，准则 57 号首批适用）**：前置部分顺序
   （封面→发行人声明→本次发行概况→目录）、「重大事项提示」位于概览之首、
   风险三分类小节结构、释义分组、「【】」留空惯例、「1-1-N」卷册页码。
3. **真实科创板申报稿（2026 年 7 月）**：新《公司法》治理表述（不设监事会、
   审计委员会履行监事会职权、「股东会」称谓）、董监高声明签字栏三分栏
   （董事/审计委员会成员/其他高管）、预披露提示措辞、律师声明新版措辞、
   前置部分新增件与顺序（声明及承诺→**致投资者声明**（上市目的/现代企业
   制度/融资必要性/持续经营，实控人签署）→发行概况→目录，卷册页码自声明
   页起算 1-1-1、封面不计页，目录含自身条目）、第十二节后的「附件一～
   十二」系列（本模板按必要性选录 6 个，`\gsannex` 宏生成节级目录条目）、
   签字版式（声明用编号标题、
   多行签名阵列＝左标签＋右侧阵列，单人/双人签名＝标签紧贴线左、整组靠右，盖章与年月日右下）；
   并按该稿实测校准了表格版式——表内五号字、行距 20.3pt（`gstab` 环境
   linespread 1.6，即 Word 表格 1.5 倍行距的观感）、正文基线距 23.3pt、
   标题层级字号/缩进/段前段后（第X节三号居中、（一）与 1、缩进两字，
   间距按实测 31/29pt 校准，注意 ctex fixskip 语义下段前段后即全部间隙）、
   前置页标题三号黑体、
   留空「【　】」括号间留半字空（`\gskong`，避免全角标点挤压成黑块）。

对照中修正过的旧准则残留（供改写时避免回退）：重大事项提示不再是目录前的
独立部分；前置「发行人声明」即扉页两段法定声明而非五段承诺；声明措辞由
「个别和连带的法律责任」改为「按照诚信原则履行承诺，并承担相应的法律责任」；
保荐人声明无先行赔付句（先行赔付作为承诺列入第九节）；律师声明不再提及
律师工作报告。
