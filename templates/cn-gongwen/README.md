# cn-gongwen — 党政机关公文模板（GB/T 9704—2012）

逐条落实 **GB/T 9704—2012《党政机关公文格式》** 的版式模板。示例内容是一份完整的
虚构下行文（示例市人民政府通知），含版头、主体、两个附件页与版记，覆盖标准
附录式样图 2（首页）、图 5/6（末页署名）、图 9（附件说明）、图 10（带附件末页）
的要素。`main.tex` 中每个排版块都注有对应条号。

## 规范落实对照

| 条号 | 规定 | 实现与实测 |
|---|---|---|
| 5.2.1 | A4；天头 37mm、订口 28mm，版心 156×225mm | `geometry`；红线实测 x 27.9–183.9mm（宽 156mm） |
| 5.2.2 | 各要素一般用 3 号仿宋 | 全文默认 `\zihao{3}\fangsong` |
| 5.2.3 | 每面 22 行、每行 28 字撑满版心 | `linespread=1.81`（225mm÷22 行 ≈ 28.98bp ÷ 16bp 推导） |
| 7.2.1–7.2.3 | 份号 6 位 3 号数字顶格第一行；密级、紧急程度 3 号黑体分行 | 零高度叠排块，增删行不影响标志定位；密级/紧急为注释开关 |
| 7.2.4 | 发文机关标志红色小标宋居中，上缘距版心上缘 35mm | 实测 72.14mm（37+35=72，误差 0.14mm） |
| 7.2.5–7.2.6 | 发文字号标志下空二行居中，六角括号；上行文字号左空一字、签发人右空一字 | 下行文为默认；上行文签发人行为注释开关 |
| 7.2.7 | 发文字号之下 4mm 与版心等宽红色分隔线 | `\nointerlineskip` 精确控制，实测基线距 4.0mm |
| 7.3.1 | 标题 2 号小标宋，红线下空二行居中，回行梯形/菱形 | 两行示例；多行行距取公文惯例 30bp |
| 7.3.2 | 主送机关顶格、全角冒号 | ✓ |
| 7.3.3 | 正文 3 号仿宋、段首空二字；层次一、黑体/（一）楷体/1.（1）仿宋 | 一、（一）为标题行（`\ctexset`，不加额外行距保持网格）；1.（1）随正文行文 |
| 7.3.4 | 附件说明：正文下空一行左空二字，名称后无标点，回行对齐 | ✓ |
| 7.3.5 | 加盖印章式：成文日期右空四字，署名以日期为准居中其上 | `tabular` 居中；印章图片留注释接口 |
| 7.3.6 | 附注居左空二字加圆括号，成文日期下一行 | ✓ |
| 7.3.7 | 附件另面编排；“附件+序号”3 号黑体顶格第一行，标题第三行居中 | 两个附件页示例（含横排表格，第 8 章） |
| 7.4 | 版记：粗 0.35mm—抄送（4 号仿宋左右各空一字、回行对齐冒号后）—细 0.25mm—印发机关左空一字/日期右空一字—末条粗线与版心下缘重合 | 末条线实测 261.7mm（版心下缘 262mm） |
| 7.5 | 页码 4 号宋体数字两侧一字线，距版心下缘 7mm；单页右空一字、双页左空一字 | `twoside` 镜像；一字线上缘实测 269.2mm（262+7） |

以上实测值为 150dpi 渲染逐像素测量结果，随仓库预览图可复核。

## 字体合规

标准用字为小标宋、仿宋、黑体、楷体。模板自动检测并优先使用正版公文字体，
未安装时回退到 ctex 按操作系统选择的近似字体，**无需任何配置即可编译**：

| 用途 | 优先使用（需自行安装） | 回退 |
|---|---|---|
| 机关标志、标题 | 方正小标宋简体（FZXiaoBiaoSong-B05S） | 宋体加粗 |
| 正文 | 仿宋_GB2312 | 系统仿宋（如 STFangsong / FandolFang） |
| 层次二楷体 | 楷体_GB2312 | 系统楷体 |

追求与机关排版完全一致时，请安装左列字体（政府办公环境通常预装）。

## 按需调整

- **份号/密级/紧急程度**：版头零高度块内按注释启用或删除（7.2.1–7.2.3）。
- **上行文（请示/报告）**：启用「签发人」注释行，删除居中发文字号行（7.2.5–7.2.6）。
- **印章**：署名块上方有 `\includegraphics` 注释接口，放置红色透明底 PNG 并微调
  位移，使其端正居中下压署名与成文日期（7.3.5.1）。
- **无附件**：删除附件说明与两个附件页，版记块移到正文页末尾。
- **可调参数**：`\gwlogodrop`（标志位置）、`\gwredrule`（红线粗细，标准未规定，
  默认 0.5mm）、`\gwlogosize`（标志字号，标准要求“醒目美观”，默认小初）。

## 偏差与边界说明

1. 无正版字体时的回退字形与机关实排存在外观差异（版式尺寸不受影响）。
2. “每行 28 字”按 LaTeX 自然排布近似（约 27–28 字）；标准 5.2.3 允许适当调整。
3. “空 N 行”类间距按正文行距（225mm/22）计，与标准 3.2 对“行”的定义
   （字高+3 号字高的 7/8）差异小于 0.4mm/行。
4. 联合行文、信函格式、命令（令）格式、纪要格式（10.1–10.3）暂未内置，可在
   本模板基础上按标准第 10 章调整。
5. 印章、签名章图片需使用者自备。
