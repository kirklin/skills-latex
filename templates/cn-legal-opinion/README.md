# cn-legal-opinion — 律所法律意见书

中国律所出具的证券法律意见书（及补充法律意见书）版式模板。框架依据
《公开发行证券的公司信息披露的编报规则第 12 号——公开发行证券的法律意见书
和律师工作报告》；版式**综合八家头部律所公开披露文书逐页实测**（对照记录见
文末），各取所长合成一套，并把差异点做成可切换的开关。

- `main.tex` — 法律意见书全本（封面＋目录＋声明＋释义＋正文六项＋签章页，10 页）
- `supplemental.tex` — 补充法律意见书（信函式信头、问询回复体例，3 页）
- `demand-letter.tex` — 律师函（催告函样张：信头＋文号＋函告事项＋盖章落款，2 页；
  与 cn-litigation 模板族为同一虚构案件，函告在前、起诉在后）
- `opinion.sty` — 版式包（三份文档共用）

```bash
bash scripts/compile.sh templates/cn-legal-opinion/main.tex --preview
bash scripts/compile.sh templates/cn-legal-opinion/supplemental.tex --preview
bash scripts/compile.sh templates/cn-legal-opinion/demand-letter.tex --preview
```

## 换成你的项目：改哪里

两份 `.tex` 开头的元信息区即全部替换点：

| 宏 | 内容 | 备注 |
|---|---|---|
| `\opFirm` / `\opFirmEn` | 律所全称中英文 | 封面题名栈、页眉、落款同步生效 |
| `\opFirmBrand` | 信头品牌短名 | 封面左上/信函信头用（如全称"北京市××律师事务所"、品牌行"××律师事务所"） |
| `\opClient` | 委托人（发行人） | |
| `\opMatter` | 事项（首次公开发行股票并在科创板上市） | 再融资/债券/并购改此行 |
| `\opConn` | 题名连接字 | `之`（竞天式）/`的`（金杜式） |
| `\opDocType` | 文书名 | `法律意见书`/`补充法律意见书（一）`…；改后同步改 `\opDocTypeSpread`（封面加宽字距版） |
| `\opRefNo` | 文号 | `××（证）字〔2026〕第0001号`（汉坤式）；置空不印 |
| `\opAddrCn/En`、`\opTel`、`\opFax`、`\opWeb`、`\opCities` | 地址块 | 封面底部与信函信头共用 |
| `\opDate` | 封面日期 | 汉字数字（二〇二六年七月），各所通行 |
| `\opFascicle` | 卷册页码前缀 | IPO 申报法律意见书卷惯例 `3-3-1` → 页脚 `3-3-1-N`；置空为纯页码 |
| `\opMatterNo` | 页脚左下斜体案号 | 通力/国际所式（`2026001/IPO/bc/D1`）；置空不印 |

签章页称谓各所有别，按需改 `\opsigblock` 标签：金杜「单位负责人」、方达/竞天
「负责人」、通力「事务所负责人」；经办律师（金杜/方达/汉坤）与承办律师（竞天）
亦为两派。红色公章与骑缝章一律用户自备透明底 PNG，钩子见 `\opSigPage` 与
`\opMakeCover` 内注释。

正文样张收录六项（批准授权/主体资格/实质条件/独立性/诉讼仲裁/结论性意见）；
完整申报用意见书按编报规则第 12 号为 23 项，全集清单见 `main.tex` 文首注释，
按同一 `\section` 体系增补即可。

## 版式从哪来：八家律所实测对照

对照样本均为公开披露文件（上交所/深交所官网及公开财经存档）：金杜（思朗科技
2026、锐成芯微 2022）、方达（蚂蚁集团 2020）、君合（联芸科技 2024）、中伦
（矽电半导体 2022 补充意见书）、汉坤（青云科技 2020）、竞天公诚（华羿微电子
2023）、海问（百奥赛图 2025 专项核查）、通力（宁德时代 2018 补充意见书）。

| 版式要素 | 各家差异（实测） | 本模板取舍 |
|---|---|---|
| 封面 | 方达：极简居中栈＋页中品牌字＋卷册号；汉坤/竞天：居中栈＋文号＋底部双语地址块；金杜：满信头（左上品牌＋右上地址块，题名栈黑体三号等大）；君合/通力：**无封面**（信函式） | **默认金杜信头式**（`\opMakeCover`：左上品牌行 `\opFirmBrand`＋右上地址块＋等大题名栈＋页底城市行；文号行为汉坤式可选项，置空即纯金杜）；竞天/汉坤对称居中式保留为 `\opMakeCoverCentered`；信函式作为 `supplemental.tex` 变体 |
| 题名连接字 | 竞天/通力用「之」，金杜/君合/中伦/汉坤用「的」 | 默认「之」，`\opConn` 一键切换 |
| 页眉 | 汉坤：左所名＋右两行「关于××公司/事项＋文书名」加细线；竞天：右侧随内容变（正文/签章页）；中伦：右上角文件类型角标；金杜：正文素面无页眉 | 汉坤式双行页眉为默认；签章页自动切「文书名签章页」（竞天式）；`\pagestyle{opplain}` 可整篇素面（金杜式） |
| 页码 | 申报卷册号 `3-3-1-N` 居中（方达/金杜/汉坤/竞天）；通力：纯页码＋左下斜体案号 `1730016/BC/pz/ewcm/D6` | 两制并存：`\opFascicle` ＋ `\opMatterNo` |
| 目录 | 汉坤设目录页（黑体条目、点线引导，仅「一、」级）；方达/金杜正文短则不设 | 收录汉坤式 `\optoc`（标题居中；`\tableofcontents` 会走一级标题样式跑成顶格，故单设命令） |
| 致语与引言 | 各家一致：顶格加粗「致：××」＋受托/依据/按照业务标准三段式 | 同业惯例照录 |
| 声明事项 | 君合：详式六段（境内法律限定/非法律专业事项/文件保证/依赖证明/用途限定）置于首页；金杜：并入引言后 | 单列「声明事项」节（君合详式内容＋金杜位置） |
| 释义 | 金杜/方达：三列全线框「术语｜指｜含义」跨页续排；文件类术语加《》 | 照录（`definitions` 环境＋`\sy`） |
| 正文层级 | 各家一致：`一、`黑体四号顶格 → `（一）`黑体小四缩进两字 → `1、(1)`宋体加粗缩进 | 照录，ctex 自动中文编号 |
| 补充意见书开篇 | **两派并存**：通力＝信函式（信头后直接行文，宁德时代 2018 补充意见书）；方达/中伦＝带封面式（方达蚂蚁合集内补充意见书自带封面＋问询问题目录，中伦矽电补充意见书有完整封面） | `supplemental.tex` 取通力信函式；要封面派用 `\opMakeCover`＋`\opDocType{补充法律意见书（一）}` 即得 |
| 问询回复（补充意见书） | 通力/中伦：反馈意见原文整段加粗，答复分层展开 | `\opFeedback{n}{…}`＋`\opReply` |
| 签章页 | 三家同式：「（本页无正文，为《全称》之签章页）」括注＋左所名（公章）＋右签名线阵列（线上手签/线下印刷体名）＋右下年月日；竞天另有「本法律意见书于××年×月×日出具」句 | 金杜/方达式为骨架（`\opSigPage`），竞天出具句为 `\opIssueLine` 可选 |
| 结论收尾 | 通力/方达：正本份数声明（「正本×份，无副本」） | `\opOriginals{肆}` |

## 版式实测记录

- 版心：A4，左右边距 70.9 pt = 25.0 mm（逐页一致），上边距约 25.5 mm
- 正文：宋体小四（12 pt），基线距 23.0 pt（Word「1.5 倍行距」实测），
  段间约 +6 pt，首行缩进两字，两端对齐；西文/数字 Times New Roman
- 一级标题：黑体四号顶格；二级黑体小四缩进两字；三级宋体加粗小四缩进两字
- 释义表、页眉线、签名线等尺寸对照公开样张目测校准
- 本模板 `\linespread{1.60}`（= 23.04 pt 基线距）、`top=26mm, bottom=25mm,
  left=25mm, right=25mm`，编译零 Overfull
