# cn-fagui — 法律法规电子文件模板（GB/T 47229.1—2026）

按 **GB/T 47229.1—2026《法律法规电子文件 第 1 部分：页面格式》** 通用页面格式
（5.2）实现的条文式文件模板。**这是法律法规正式文本的发布排版**——面向立法、
规章与司法解释发布、制度条文汇编等专门场景；日常法律文书（合同、起诉状、
法律意见书等）不适用本格式。
页面底座（版心 156mm×225mm、三号仿宋 22 行/28 字、页码一字线）与 GB/T 9704
一致，与 `cn-gongwen` 同源；增量是法律文本特有的结构层级与字体规则。
`main.tex` 中每个排版块都注有对应条号。

## 规范落实对照

| 条号 | 规定 | 实现 |
|---|---|---|
| 4.2 | A4；上 37mm、左 28mm，版心 156×225mm；3 号仿宋 22 行/28 字 | `geometry` + `linespread=1.81`（推导同 cn-gongwen） |
| 5.2.2 | 标题 2 号小标宋，首页空两行 3 号字位置，居中，多行行距 1cm | 实测标题上缘 57.4mm（37+2 行），块内 `\baselineskip=1cm` |
| 5.2.3 | 题注：标题下空一行，3 号楷体，左右各空二字、首行再空二字，圆括号 | `\leftskip`/`\rightskip`/`\parindent` 组合 |
| 5.2.4 | 发文字号（司法解释）：题注下空一行居中，六角括号 | 注释开关 |
| 5.2.5 | 目录：3 号楷体，显示到节；编/章左空二字、节左空四字；「目　　录」二字间空两全角 | 示例含目录节选 |
| 5.2.6 a | 编/分编/章：第×编 中文数字，3 号黑体居中，上下各空一行 | `\fabianX`/`\fafenbianX`/`\fazhangX`，相邻标题行的空行用 `\addvspace` 合并 |
| 5.2.6 b | 节：第×节，3 号宋体居中，每章下从一起排 | `\fajieX`（计数器随章重置） |
| 5.2.6 c | 条：第×条 全文连续中文数字，3 号黑体，左空二字，空一全角接条文 | `\fatiaoX`（`\chinese` 自动生成如「第四百二十五条」） |
| 5.2.6 d–f | 款：无编号自然段；项：（一）；目：1.（3 号半角） | 直接以自然段书写（见示例） |
| 5.2.7 | 页码：4 号宋体一字线，距版心下缘 7mm，单右空一字/双左空一字 | `twoside` 镜像，同 cn-gongwen 实测 |

## 文件结构

| 文件 | 作用 |
|---|---|
| `main.tex` | 唯一源文件 — 版式参数 + 结构宏 + 《民法典》真实条文节选示例 |

示例内容为《中华人民共和国民法典》真实条文节选（总则编第一至三条、物权编
第四百二十五至四百二十七条）——法律条文依《著作权法》第五条不受著作权保护，
标准附录示例亦采用民法典条文。

## 改成你的文件

1. 标题与题注：替换标题文字与题注内容（制定/修改/批准信息，各项之间空一个
   全角字符）；司法解释需要发文字号时启用注释行。
2. 结构宏按层级书写：
   - `\fabianX{总\hspace{2\ccwd}则}` / `\fafenbianX{…}` / `\fazhangX{…}` /
     `\fajieX{…}` — 序号自动生成；两字标题用 `\liangzi{总}{则}` 空出二全角；
   - `\fatiaoX 条文内容…` — 条号全文连续自动编号；
   - 款直接另起自然段；项写「（一）…；」；目写「1.…」；
   - 节选/汇编场景用 `\setcounter{fatiao}{424}` 等对位到起始序号（见示例）。
3. 目录按需保留：楷体，编/章左空二字、节左空四字，仅列到节。
4. 修正案、修改决定等可改用「一、（一）1.（1）」层次（5.2.6 末段），
   字体规则同 GB/T 9704（黑/楷/仿/仿）。

## 编译

```bash
bash scripts/compile.sh templates/cn-fagui/main.tex --preview
```

字体自动检测与回退策略同 `cn-gongwen`（方正小标宋 / 仿宋_GB2312 /
楷体_GB2312 装了就用，否则按操作系统回退），无需配置即可编译。

## 边界说明

1. 行政规范性文件页面格式（5.3，标题+发文字号+主送+署名结构）与 GB/T 9704
   公文主体一致，可参照 `cn-gongwen` 排版。
2. 相关资料（附录 A：令、公告、草案的说明、修改情况汇报、审议结果报告、
   修改意见报告）暂未内置；令/公告可参照 `cn-gongwen/mingling.tex` 调整。
3. 表格（第 7 章：表题 3 号黑体居中于表上）与图（第 9 章）未在示例中演示，
   按条款自行添加即可。
