# cn-letter — 中文书信

正式中文书信的通用骨架，附两个最常用的样张：

| 文件 | 文书 | 什么时候用 |
|---|---|---|
| `resignation.tex` | 辞职信（高管版：CEO 致董事会） | 提出离职。含《劳动合同法》第三十七条法定要素＋高管善后全套（交接清单/离任审计/保密竞业/信息披露）；普通员工删繁即可 |
| `cover-letter.tex` | 求职自荐信（高管版：应聘 CEO） | 投递自荐。三段式匹配点全用可核验数字；普通岗位同构换事实，与 `templates/resume` 配套 |

共用 `letter.sty`。编译：

```bash
bash scripts/compile.sh templates/cn-letter/resignation.tex --preview
```

## 版式

标准中文书信五段式：居中标题（黑体二号）→ 称谓顶格（尊敬的××：）→
正文（宋体小四、1.5 倍距、首行缩进两字）→ 信末礼语（**「此致」空两格、
「敬礼！」另起行顶格**——最容易写错的地方，宏已固定）→ 右下落款
（署名＋日期）。单页无页眉页脚。

宏：`\ltTitle` 标题、`\ltDear` 称谓、`\ltClosing` 此致敬礼、`\ltSign`
落款行（可多行连用）、`\hl` 加粗。

## 扩展成其他信

感谢信、推荐信、介绍信、道歉信都是同一骨架：复制任一样张，换标题、
称谓与正文即可。求职自荐信的写法要点（每段一个能力主题、用事实不用
形容词、控制一页）见 `cover-letter.tex` 头注。
