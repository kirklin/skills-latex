# cn-litigation — 诉讼与仲裁文书（民事·劳动·刑事）

律师实务中法院/仲裁委面向的常用文书模板族，覆盖民事、劳动、刑事三个方向：

| 文件 | 文书 | 领域 | 版式依据 |
|---|---|---|---|
| `complaint.tex` | 民事起诉状（民间借贷纠纷，3 页） | 民事 | 最高法、司法部、全国律协《部分案件起诉状答辩状示范文本》（法〔2025〕82 号，2025-07-14 起全面推行，67 类）——表格化、要素式、勾选式 |
| `answer.tex` | 民事答辩状（2 页） | 民事 | 同上（答辩事项逐项对应诉讼请求表态） |
| `labor-arbitration.tex` | 劳动人事争议仲裁申请书（2 页） | 劳动 | 北京市劳动人事争议仲裁委员会官方表格（当事人信息/请求事项/事实和理由/证据清单/五条填表注意事项） |
| `defense-statement.tex` | 刑事辩护词·一审（2 页） | 刑事 | 无官方格式；对照刑辩专业所及司法行政机关公开发布的辩护词通行体例 |
| `litigation.sty` | 共用版式包 | — | 两套底座：要素式表格文书 ＋ 行文式文书 |

```bash
bash scripts/compile.sh templates/cn-litigation/complaint.tex --preview
# answer / labor-arbitration / defense-statement 同理
```

配套：律所信纸类的《律师函》在 `templates/cn-legal-opinion/demand-letter.tex`
（与本模板族共用同一虚构案件：函告在前、起诉在后，日期金额两处一致）。

## 版式要点（litigation.sty）

- **要素式表格文书**：标题宋体加粗二号居中＋案由副题；通栏细线「说明」框
  （含《民事诉讼法》第十三条诚信原则特别提示，★ 已声明入 CJK 字类防缺字）；
  左「项目」右「内容」全线框要素表（`liform` 环境，`\lirow`/`\lisec`/
  `\lispan`/`\likv`）；自绘勾选框 `\ckbox`/`\ckboxv`（不依赖字体 □/☑ 字位，
  同 cn-ipo 实现）；法院文书「— n —」一字线页码
- **行文式文书**：称呼语顶格（`\liVocative`）、「一、」宋体加粗顶格分点
  （ctex 自动中文编号）、`\liSubmitTo{××人民法院}`（此致缩进两字＋机关名
  另行顶格）、`\liSign` 右区落款
- 正文宋体小四、1.5 倍行距、首行缩进两字；表格内五号

## 对照校验记录

1. **民事起诉状/答辩状**以法〔2025〕82 号现行版为准。参照原件：广州番禺法院
   公开的民间借贷纠纷起诉状示范文本（2024 版表格）＋ 法〔2025〕82 号通知
   原文（驻马店中院公开 PDF）。2025 版相对 2024 试行版的三处变化已实现：
   **删除「送达地址」「电子送达」栏目**；「诉讼请求」「事实与理由」项下
   **增加空白补充栏**（`\lispan{其他补充：…}`）；**新增「对纠纷解决方式的
   意愿」先行调解栏目**。说明框措辞、当事人信息要素（自然人/法人两式、
   法人类型勾选清单）、民间借贷请求要素（本金/利息/实现债权的费用/诉讼
   费用）照录官方文本。
2. **劳动仲裁申请书**：要素项照录北京市官方 docx 表格（姓名/户口性质/
   统一社会信用代码/劳动合同履行地/是否签合同/单位联系人等），官方原表为
   申请人、被申请人左右双栏，此处按模板族统一的左项目右内容式改排；
   文末五条填表注意事项照录。各地仲裁委表格大同小异，按当地表微调即可。
3. **辩护词**：结构对照公开发布的辩护词实例（标题＋案件引题／尊敬的审判长、
   审判员／受托与阅卷段／分点意见／综上所述与量刑建议／此致／辩护人落款）；
   样张所引法条（《刑法》第六十七条、第七十二条，《刑事诉讼法》第十五条、
   第三十七条，自首司法解释，量刑指导意见）均为现行有效条文。
4. 样张数据自洽：借贷案借款 200,000 − 已还 20,000 = 尚欠 180,000（答辩人
   另主张已还 30,000 属争议点）；劳动案月薪 8,200×3 个月欠薪 = 24,600，
   工作年限 3 年 3 个月 → 补偿基数 3.5 个月 → 违法解除赔偿金 2 倍 =
   7 个月×8,200 = 57,400。

## 换成你的案件：改哪里

- 案由：改 `\liTitle` 副题＋「诉讼请求和依据」「事实与理由」的要素行
  （67 类示范文本均为同一框架，要素行按当地法院公布的对应案由文本增删）
- 当事人：自然人/法人两种行式都已给出，多被告/第三人按同式增行
- 勾选：`\ckbox` 未选、`\ckboxv` 选中；填写值直接写在 `\likv{标签}{值}` 里
- 辩护词：每个 `\section` 即一个辩护要点，小标题写结论句
